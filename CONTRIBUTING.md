# Code Base Setup

## Clone the Github Repostiory

```shell
git clone https://github.com/Harry-Hopkinson/create-solid-app
```

## Install the Dependencies

### Yarn

```shell
yarn install
```

### NPM

```shell
npm install
```

## Build and Run the Application

### Yarn

```shell
yarn build && yarn start
```

### NPM

```shell
npm run build && npm run start
```
